# 办公员工管理系统 python632

## 项目概述

办公员工管理系统是基于 Python 语言与 MongoDB 数据库开发的，旨在提供高效便捷的办公管理解决方案。

## 技术栈

- **后端语言**：Python
- **框架**：Flask
- **数据库**：MongoDB
- **开发工具**：PyCharm

## 功能模块

系统实现了以下功能模块：

- **报表审批**
- **日程管理**
- **人员管理**
- **公告分布**
- **站内信息**
- **请假申请**
- **会议申请**
- **请假审批**
- **会议审批**

## 系统角色

- **管理员**：负责系统的维护与管理。
- **员工**：使用系统的各项功能。

## 运行环境

- Python 环境配置要求
  ```plaintext
  Python 3.x
  ```
- MongoDB 环境配置要求
  ```plaintext
  MongoDB 4.x
  ```

## 部署步骤

1. 确保环境中已安装 Python 和 MongoDB。
2. 克隆或下载项目代码。
3. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
4. 配置数据库连接。
5. 运行应用：
   ```bash
   python run.py
   ```
